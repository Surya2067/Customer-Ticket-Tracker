# Oracle APEX - Customer Ticket Tracker App

This is a simple APEX + PL/SQL demo app to manage and track customer service tickets.

## Features
- Customer & Ticket management
- Real-time ticket tracking
- Basic PL/SQL for ticket updates

## Setup
1. Run `schema.sql` to create tables.
2. Run `sample-data.sql` to insert demo data.
3. Import `apex_app_export.sql` into Oracle APEX.
4. Compile `ticket_pkg.sql` in SQL Workshop.
